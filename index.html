What do we provide :

Password cracking uses a set of commands. To make stuff simpler we provide this interface.

HashCat : 300+ Hash-types implemented

Docs :

You could choose your method of input :

  • Upload the file
  • Type in the hash

Once you press the submit button we would process your data in the backend and return a response as provided by using hashcat on our server

Working :

  • We process the input from the user by running "hashcat -a <-code of hash-> -m 0 <-hashfile-> <-dictionary->" command and the result is then streamed in and outputted on your screen

  • If there is no output then most likely hashcat wasn't able to brutefore it so check if the hash type and corresponding hash entered is correct

  • If there is no error on the user's side, please click on the issue button below and mention the issue in the new window opened
